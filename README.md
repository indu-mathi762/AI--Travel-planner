
# 🌍 AI Travel Planner ✈️

> **Your Next Adventure, Engineered by AI**

An AI-powered travel planning assistant that creates personalized travel experiences based on user preferences. The application generates destination suggestions, day-wise itineraries, budget estimates, and packing checklists to make trip planning simple, fast, and enjoyable.

## 📌 Project Overview

Planning a trip often involves researching destinations, estimating costs, creating itineraries, and preparing packing lists. This AI Travel Planner automates the entire process by generating customized travel plans in just a few clicks.

Users simply enter their travel preferences, and the application provides an intelligent travel plan tailored to their needs.

## 🎯 How It Works

1. Enter your starting location.
2. Select your travel month.
3. Choose the number of travel days.
4. Set your budget.
5. Select accommodation and transportation preferences.
6. Click Generate.
7. Receive:
   - Personalized destination suggestions
   - Day-wise itinerary
   - Budget estimate
   - Packing checklist
